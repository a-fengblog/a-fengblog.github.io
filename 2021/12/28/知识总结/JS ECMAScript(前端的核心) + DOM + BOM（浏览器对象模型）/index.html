2.网址

例如:百度：https://www.baidu.com -> 百度的首页 ⭐

URL（Uniform Resource Locator）（网络资源统一定位符 -> 表示一个文件地址）：

协议+主机地址(ip/域名)+端口号+路径?+文件名?+参数?+锚点?

协议:http:// https:// 网络传输协议 -> 让别人的电脑访问分享的文件

主机地址：IP地址 -> 域名（易记 -> 通过DNS解析域名找到IP）

域名：服务器名 + 主域（qq.com / 163.com …)

本地打开服务：

​ 主机IP：127.0.0.1 => localhost 只有自己可以访问的服务形式

​ 局域网：192.168.124.39

​ 无线网：查看自己电脑输入ipconfig 192.168.124.1本机

端口号：可以理解为主机打开的窗户 -> 不能跨域窗口访问其它内容

http 端口号为 :80

https 端口号为 :443

— 上述三者共同组成 源(origin) = 协议 + 主机地址 + 端口号

​ 例如: http://127.0.0.1:5500

— 如果三者完全一致，称为同源，任意一个不一致，就非同源

路径：访问窗口分享的文件夹下的其它文件内容

-> 文件名

-> 空路径（路由：后台拦截路劲，定义返回内容）

参数：?name=value&name=value&…

锚点：#abc

3.open方法

相当于js中获得A标签的能力

传入三个参数，第三个参数是一个逗号分隔的设置字符串，表示在新窗口中都显示哪些特性。下表列出了可以出现在这个字符串中的设置选项。

设置 值 说明
fullscreen yes或no 表示浏览器窗口是否最大化。仅限IE
height 数值 表示新窗口的高度。不能小于100
left 数值 表示新窗口的左坐标。不能是负值
location yes或no 表示是否在浏览器窗口中显示地址栏。不同浏览器的默认值不同。如果设置为no，地址栏可能会隐藏，也可能会被禁用（取决于浏览器）
menubar yes或no 表示是否在浏览器窗口中显示菜单栏。默认值为no
resizable yes或no 表示是否可以通过拖动浏览器窗口的边框改变其大小。默认值为no
scrollbars yes或no 表示如果内容在视口中显示不下，是否允许滚动。默认值为no
status yes或no 表示是否在浏览器窗口中显示状态栏。默认值为no
toolbar yes或no 表示是否在浏览器窗口中显示工具栏。默认值为no
top 数值 表示新窗口的上坐标。不能是负值
width 数值 表示新窗口的宽度。不能小于100

如果第二个参数设置后效果是打开新的标签页，再加上此参数，会打开新的浏览器窗口；

如果第二个参数设置后效果是不打开新标签页，方法会忽略第三个参数的作用：

4.history: 历史记录对象

保存从窗口打开后，成功访问过的url的历史记录栈。出于安全方面的考虑，开发人员无法得知用户浏览过的 URL。不过，借由用户访问过的页面列表，同样可以在不知道实际 URL 的情况下实现后退和前进。

history + length 当前标签页历史记录的长度

history + go(n) 跳转n页（正向前，负向后）

history go(0) 刷新当前页

history + forward() 向前跳转一页

history + back() 向后跳转一页

7.正则表达式

正则：规则表达式，用于匹配字符串中某段内容，对象类型

声明：

1. new RegExp(规则, 属性)

2. /规则/属性

规则：描述匹配的内容

属性：i 忽略大小写 g 全局匹配 m 多行匹配/ u开启unicode支持 / y精准匹配

​ 属性不论顺序

使用：

正则.test(字符串) 返回字符串是否有符合规则的片段 true/false

字符串.match(正则) 返回一个数组，内容是 字符串中符合规则的片段，如果没有返回 null（正则带 g(全局匹配) 匹配所有符合条件的)
