一.FLEX.html,弹性布局

弹性布局：flex

给元素设置 diaplay: flex / inline-flex; 元素就会成为一个弹性容器

设置为 diaplay: flex 块元素特征

+ 容器默认有两个轴线：

- 默认水平方向的 主轴 - 控制直接子元素的水平排列方式

- 默认垂直方向的 交叉轴 - 控制直接子元素的垂直排列方式

+ 弹性容器的《直接子元素》称为弹性元素

- 会默认沿着主轴方向排列

- 换行后遵循 交叉轴设置的排列方式

- 宽高生效

二.容器的样式

justify-content

justify-content:flex-start;//控制弹性元素在主轴的排列方式

flex-start 起始位置

center 居中中

flex-end 结尾位置

space-around 空白环绕 - 元素两侧空白间隙相等;(左右间隙等分)

space-between 两侧元素贴容器，元素之间的空白间隙相同;

space-evenly存在兼容 空白间隙等间分布; (空白间隙等分)

align-items

align-items: flex-start; 《单行》弹性元素的交叉轴排列方式

flex-start flex-end center baseline

stretch 当弹性元素没有交叉轴对应方向的值(宽/高)时，设置对应宽高 = 交叉轴高度 / 行数

align-items: stretch;

flex-wrap

​ flex-wrap: nowrap 控制弹性元素的换行

​ nowrap wrap wrap-reverse（沿着交叉轴的中线进行翻转）

​ flex-wrap: wrap-reverse;

align-cotnent

​ align-cotnent: 多行弹性元素在交叉轴的整体布局

​ flex-start flex-end center space-around space-between space-evenly stretch

​ align-content: space-between;

flex-direction

​ flex-direction: row 设置弹性容器主轴方向

​ row 从左到右

​ row-reverse 从右到左

​ column 从上到下

​ column-reverse 从下到上

​ flex-direction: column;

flex-flow:简写

​ flex-flow: flex-wrap flex-direction; 简写，wrap direction可以换位置

​ flex-flow: wrap-reverse column;

flex-flow: row-reverse wrap;

三.弹性元素样式.html

flex-basis: 定义项目占据的主轴空间，内容原始大小>权重>宽高设置

​ flex-basis: 25px;

flex-grow: 当主轴空间有剩余时，定义项目的放大比例，默认为0，表示不放大

​ flex-grow: 1;

flex-shrink：当主轴空间不足时，定义项目的缩小比例，默认为 1，为0表示不缩小

​ flex-shrink: 0;

​ flex: flex-grow flex-shrink flex-basis; 简写形式

​ flex: flex-basis flex-grow flex-shrink;

​ flex: none; => flex: 0 0 auto;

​ flex: auto; => flex: 1 1 auto;

​ flex: 6; => flex: 6 1 0%;

​ flex: 6 4; => flex: 6 4 0%;

​ flex: 6 4px; => flex: 6 1 4px;

​ flex: 4px 9; => flex: 9 1 4px;

​ flex: 20%; => flex: 1 1 20%;

order: 定义项目的排列顺序，值越大，越靠后，默认为 0

四.margin在flex

flex容器中，各个方向的长度已经是已知的，因此 margin: auto 可以均分对应方向的空白空间

margin会占用空白空间，导致 justify-content / align - * 布局失效
