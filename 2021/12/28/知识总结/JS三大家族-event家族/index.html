元素拖拽

  • 逻辑一: 元素移动的位置 = 元素原始位置 + 鼠标移动的距离 鼠标移动的距离 = 移动过程中鼠标与页面的距离 - 按下时鼠标与页面的距离

  • 逻辑二: 鼠标与页面的距离 = 鼠标与元素的距离 + 元素与页面的距离

    • 示例:

    • ```js
      var content = document.querySelector(‘.content’);

      var box = document.querySelector('.box');

box.onmousedown = function (e) {  //鼠标按下时
  var x = e.pageX - box.offsetLeft,  //元素移动的位置 = 元素原始位置 + 鼠标移动距离
    //鼠标移动距离 = 移动过程中鼠标与页面的距离 - 按下鼠标时鼠标与页面的距离
    y = e.pageY - box.offsetTop;  // 此时的数据是鼠标按下时的元素与视口距离 (固定的)
  window.onmousemove = function (e) { //鼠标移动中
    //这里用 : 鼠标与页面的活动的距离 - 上面固定的距离 == 这个元素的活动距离 
    var moveX = e.pageX - x;  //  X
    var moveY = e.pageY - y;  //  Y
    var ww = content.offsetWidth - box.offsetWidth; // 元素活动的范围
    var hh = content.offsetHeight - box.offsetHeight; //大盒子的宽度-小盒子的宽度
    console.log(ww, hh);
    if (moveX < 0) {
      moveX = 0
    }
    if (moveX > ww) {
      moveX = ww
    }
    if (moveY < 0) {
      moveY = 0
    }
    if (moveY > hh) {
      moveY = hh
    }
    box.style.left = moveX + 'px';
    box.style.top = moveY + 'px';
  }
}
window.onmouseup = function () {//鼠标松开时
  window.onmousemove = null; // 鼠标移出视口时
}

element.colsest()

  • element.colsest() 方法: 匹配特定选择器且离当前元素最近的父类元素(也可以是当前元素本身);如果匹配不到,则返回 null 值;IE不支持此方法

  • 语法 : var closestElement = targetElement.closest(selectors);

  • 当事件委托中有子元素时: 可以通过匹配特定的元素属性添加事件 示例:

  • var lis = document.querySelectorAll("li"),
      ulEle = document.querySelector("ul"),
      btn = document.querySelector("button");
    btn.onclick = function (){
      var liEle = document.createElement("li");
      liEle.innerHTML = "<p>LI_" + (ulEle.children.length + 1) + "</p>";
      ulEle.append(liEle);
    }
    // 改为事件委托 -- 要判断触发事件的元素
    ulEle.onclick = function (e){
      // element.tagName 返回元素的标签名，标签名是大写的
      // element.closest(selector) 返回距离element元素最近的符合selector选择条件的父元素，包括自己。如果没找到返回 null
      // 用了 closest 只有两个结果：1. 找到了元素  2. null
      // console.log(e.target.tagName);
      var ele = e.target.closest("li");
      if(ele){
        ele.style.color = "#f00";
      }
      // console.log(e.target);
    }
