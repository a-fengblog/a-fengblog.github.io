html5-css3新增

1.元素可编辑属性

contenteditable="true"

在body中给元素添加行内样式,元素就可以被编辑

在js中可以开启全文编辑

2.全屏属性

想让元素变成全屏后添加的伪类元素 css样式

:fullscreen 设置全屏状态样式的伪类 – 全屏状态下，元素的宽高位置属性无法更改

如果写兼容，要这样写 :-webkit-full-screen

3.拖拽属性

给元素添加 draggable="true"，元素就会变成拖拽元素，可以被拖拽出一个快照，具有以下方法：

+ ondragstart 开始拖拽 ⭐

e.dataTransfer.setData(key, value) 传值设置

+ ondrag 拖拽持续中

+ ondragleave 鼠标离开拖拽元素本体

+ ondragend 拖拽结束

与拖拽元素有交互的元素 – 目标元素，具有如下事件

+ ondragenter 鼠标进入目标元素

+ ondragleave 鼠标离开目标元素

+ ondragover 鼠标在目标元素中停留 – 页面禁止元素堆叠，因此会阻止drop事件执行，如果希望drop可以执行，需要在 over事件中阻止默认事件 ⭐

+ ondrop 鼠标在目标元素中释放拖拽元素 ⭐

e.dataTransfer.getData(key) 获取值

4.本地存储

传统方式我们以 *document.cookie* 来进行存储的，但是由于其存储大小只有4k左右，并且解析也相当的复杂，给开发带来诸多不便，HTML5规范则提出新的解决方案 – localstorage 和 sessionStorage，IE8 以上的 IE 版本支持这两个属性。

存储的两种方式：(面试) ⭐⭐⭐

localStorage: 本地存储

+ 值一旦存储，用户不删除，一直存在

+ 同源页面共享

sessionStorage: 临时存储

+ 存储的内容会跟着页面关闭被清空

+ 每个页面都有自己的临时存储，不共享

二者使用的属性一致：

+ setItem(key,value) 设置存储内容

+ getItem(key) 获取存储内容

+ removeItem(key) 删除存储内容

+clear() 清空存储内容

+ length 存储内容的长度

+ key(index) 根据下标存储的 键名(key)

​ key方法 + length 组合使用获取存储的所有内容

不推荐的操作：

通过 localStorage / sessionStorage 直接获取/设置 存储的键值

localStorage.key = value

查看存储的内容：

开发者工作台 -> Application -> 左侧栏 Storage -> Local / Session

注意：

  1. 存储的顺序不是取值的顺序，在存储中有自己的规则给存储内容排序

    1. 重复给同一个key存储不同的值，后面的值会覆盖前面的

    2. 引用类型存储 – 将其变成 JSON 进行转换

同源：协议 + 主机地址 + 端口号，三者完全一致就是同源

5.非选择器

:not() 伪类可以将一个或多个以逗号分隔的选择器列表作为其参数。选择器中不得包含另一个否定选择符或伪元素。

注意：以逗号分隔的选择器列表作为参数是实验性的，尚未获得广泛支持。

6.计算样式

calc

calc(expression) 函数用于动态计算长度值

  • expression 必须，一个数学表达式，结果将采用运算后的返回值。

  • 需要注意的是，+ - 运算符前后都需要保留一个空格(尽管 * / 不加也能用)

  • 计算方式遵循数学表达式优先级，先括号再乘除再加减

  • 带单位的可以互相加减，数字与单位可以乘除

  • 此函数可以应用于任何表示长度的位置，CSS时间单位也算是长度值

7.css变量

在CSS中定义变量，它的值可以是任意CSS样式值，所有使用该变量的地方，在变量发生改变后，都会跟着改变

定义变量：

--变量名: CSS样式值

+ 可以定义在任意位置，一般来说全局位置会定义在 :root 伪类中(全局变量)

+ 变量可以重复定义，元素在使用 变量值 时优先使用自己定义的，然后是父级的，往上直到全局

+ 同一个元素在不同选择器中定义相同的变量，遵循 CSS选择器优先级

使用变量：

var(--变量名, 默认值);

默认值在 变量名未声明/值为 initial 时生效

变量值为空，仍是一个有效值，并不会使用默认值

在变量值与设置的样式不一定或者是一个空值时，使用改样式的原始值

不推荐的用法：

var(–a,var(–b, var(–c, …)))

JS操作CSS变量

getComputedStyle(ele,null).getPropertyValue(“–变量名”);

ele.style.setProperty(“–变量名”, “值”);

8.动画

动画通过 @keyframes 指定动画序列，通过百分比将动画序列分割成多个节点，在各节点中分别定义各属性。然后通过 animation 将动画应用于相应元素。

关键属性

  • animation-name 动画名称(必填)
  • animation-duration 动画持续时间(必填)
  • animation-timing-function
    • linear/ease/ease-in/ease-out/ease-in-out/cubic-bezier(n,n,n,n)： 特定的贝塞尔曲线类型
  • animation-delay 动画延迟（只是第一次）
  • animation-iteration-count 重复次数
    • 1/2/3/4…
    • infinite 无限次
  • animation-direction 动画是否重置后再开始播放
    • normal 动画每次都从 0% 的状态开始执行
    • alternate 动画从起点开始，往复运动
    • alternate-reverse 动画从终点开始，往复运动
  • animation-fill-mode 动画执行完毕后状态
    • forwards 当动画完成后，保持最后一个属性值（在最后一个关键帧中定义） - 未实现，当前效果类似于 both。
    • backwards 在 animation-delay 所指定的一段时间内，在动画显示之前，应用开始属性值（在第一个关键帧中定义）。
    • both 设置对象状态为动画结束或开始的状态，结束时状态优先

简写语法

name duration 是必填属性，其它的属性是都可选属性。

动画状态

animation-play-state 动画状态（running 执行 和 paused 暂停）

帧动画

在应用 CSS3 动画时，有个控制时间的属性 timing-function 。它的取值中除了常用到的 贝塞尔曲线 以外，还可以是 steps() 函数

steps(n,start/end)

第一个参数 number 为指定的间隔数，即把动画分为 n 步阶段性展示

第二个参数默认为 end，设置最后一步的状态，start 为结束时的状态，end 为开始时的状态(不用记,第二个参数不写就行)

注意: n 必须是正整数

动画的事件：
